# RESPEC Template – Nationaal Archief

Deze repository bevat een gestandaardiseerde RESPEC-template die gebruikt kan worden door het Nationaal Archief voor het opzetten van nieuwe kennisproducten in GitHub.

## Doel

Deze template biedt een snelle en consistente manier om een nieuwe repository op te starten. Door gebruik te maken van een vaste structuur en vooraf ingerichte bestanden kunnen teams direct aan de slag met het ontwikkelen en documenteren van een kennisproduct.

## Wat biedt deze template?

- Een basisstructuur voor RESPEC-documentatie
- Vooraf ingestelde configuratiebestanden
- Richtlijnen voor het opbouwen van een kennisproduct
- Consistente opzet voor samenwerking binnen en tussen teams
- Snelle start zonder telkens opnieuw alles in te richten
